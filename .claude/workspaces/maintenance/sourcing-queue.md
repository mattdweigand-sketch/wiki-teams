# Sourcing Queue

Open asks for new source material. The maintenance workspace uses this to prioritize which `raw/` documents to chase.

## How to add an entry

```
## <slug> — <one-line description of what's missing>

**Why we need it:** which pages or questions are blocked or under-sourced.
**Where to look:** team member, system, vendor, public doc — whatever points at the right place.
**Priority:** high | medium | low
**Added:** YYYY-MM-DD
```

When an item is sourced, move it out of "Open" into "Resolved" with a one-line note about which source filled the gap.
