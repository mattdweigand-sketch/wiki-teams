# Contradictions

Tracking ground for places where sources disagree. Pages affected by an open contradiction should be marked `confidence: contested` and link back here.

## How to add an entry

```
## <slug> — <one-line summary of the disagreement>

**Status:** open | resolved | superseded
**Pages affected:** [[page-slug]], [[other-page]]
**Sources in tension:** [[source-a]] vs. [[source-b]]

**The disagreement:**
- Source A says ...
- Source B says ...

**Resolution (if any):**
- ...
```
